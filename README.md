# Inclusive Lecture Screen Reader

## Overview
An AI-powered accessibility tool that converts classroom board and slide content into speech for visually impaired students using OCR and text-to-speech.



## Problem Statement
Visually impaired students often struggle to access information written on classroom whiteboards, projected slides, or handwritten notes in real time.

This project aims to improve accessibility and inclusive education by converting visual classroom content into spoken audio using computer vision and AI.
